# Metorial Voyager Client

A client library for interacting with the Metorial Voyager platform using TypeScript.

## Usage 

```ts
import { createVoyagerClient } from '@lowerdeck/voyager-client';

let voyager = createVoyagerClient({
  endpoint: '...'
});
```
